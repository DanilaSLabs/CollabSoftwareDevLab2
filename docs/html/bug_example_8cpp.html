Detailed Description

A little with logical bugs in it.

This program is littered with logical bugs. Your task is the following: 1) Compile and run the program. 2) For each bug you find you must fix the bug then commit the code with a comment on the fix you did 3) There are a four functions that are not completed, you need to complete them and also update the menu and the main function
